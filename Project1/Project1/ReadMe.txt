Για να τρέξει κάποιος αυτήν την εφαρμογή πρέπει αρχικά να δημιουργήσει την βάση δεδομένων απο το dataBase.sql αρχείο.
Έπειτα, πρέπει να πάει στο αρχείο DataBaseAccess.java που βρίσκεται στο MainApplication\src\mainapplication  και να
αλλάξει στις γραμμές 29, 30 το USER, PASS  βάζοντας τα δικά του username, password για την mySQL.
Εν συνεχεία, πρέπει να κάνει ένα clean and build project μέσα απο το NeatBeans για να πάρει τις αλλαγές.
Τέλος, αρκεί να τρέξει το jar αρχείο που βρίσκεται στο MainApplication\dist απο το cmd.

Τα αρχεία που κρατάνε τίς συναλλαγές αποθηκεύονται στο directory(cmd) απο το οποίο βρίσκεται ο χρήστης όταν τρέχει την
εφαρμογή.
